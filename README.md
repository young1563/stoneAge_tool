# StoneAge Idle Gacha System Analysis Tool

데이터 기반 게임 시스템 분석 포트폴리오

---

# 1. 도구 개발 배경 (Background)

모바일 게임 시장에서 **확률형 시스템(Gacha System)**은 캐릭터, 장비, 펫 등의 수집 콘텐츠를 중심으로 널리 사용되는 핵심 시스템이다.
특히 **Idle RPG, 수집형 RPG, 캐주얼 RPG 장르**에서는 가챠 시스템의 확률 설계가 유저 경험과 게임 경제 구조에 직접적인 영향을 미친다.

그러나 실제 게임에서 공개되는 확률표는 대부분 **단순한 수치 데이터 형태**로 제공되며, 다음과 같은 한계가 존재한다.

* 확률 구조를 직관적으로 이해하기 어렵다
* 유저가 체감하는 실제 확률을 분석하기 어렵다
* 확률 시스템이 게임 경제 및 과금 구조에 미치는 영향을 파악하기 어렵다

또한 최근 모바일 게임의 가챠 시스템은 다음과 같은 다양한 설계 요소가 결합되어 있다.

* 픽업 확률 시스템
* 진행도 기반 확률 변화
* 이벤트 확률 보정
* 천장 시스템(Pity System)

이러한 시스템은 단순 확률표만으로는 구조를 이해하기 어려우며,
**데이터 분석과 시뮬레이션을 통해 구조를 분석할 필요가 있다.**

---

# 2. 도구 활용 목적 (Purpose)

본 **분석 도구**는 모바일 게임 **“스톤에이지 키우기”의 확률 데이터를 기반으로 가챠 시스템의 구조를 분석하고 시뮬레이션하는 데이터 기반 분석 도구이다.**

이를 통해 다음과 같은 질문에 대한 분석을 수행한다.

* 가챠 시스템의 확률 구조는 어떻게 설계되어 있는가
* 특정 희귀 캐릭터를 획득하기까지 평균적으로 몇 번의 뽑기가 필요한가
* 유저가 체감하는 실제 획득 확률은 어느 정도인가
* 픽업 시스템은 실제 확률에 얼마나 영향을 미치는가
* 가챠 시스템은 게임 경제 구조에 어떤 영향을 미치는가

본 도구에서는 **Python 기반 데이터 분석과 Monte Carlo 시뮬레이션을 활용하고, Streamlit을 통해 인터랙티브 분석 대시보드를 구현한다.**

# 3. 주요 기능 (Key Features - v1.0)

현재 분석 도구는 다음과 같은 5가지 핵심 메뉴로 구성되어 있습니다.

### 📊 1. 확률 구조 분석
- 등급별(SS, S, A, B, C) 기본 등장 확률 데이터 시각화
- 일반 가챠 대비 픽업 가챠의 효율성 비교
- **레벨 기반 확률 보정:** 플레이어 레벨 성장에 따른 SS 등급 확률 상향 곡선 분석

### 🎲 2. 가챠 시뮬레이터 (Monte Carlo)
- 사용자 정의 시행 횟수(n) 및 반복 횟수(Trials) 기반 시뮬레이션
- 획득 분포 히스토그램을 통한 '운의 영역(Variance)' 시각화
- 대량 시뮬레이션 데이터를 통한 실제 기대 확률 검증

### 📈 3. 유저 체감 확률 분석
- 독립 시행 원리에 따른 **누적 획득 확률($1 - (1 - p)^n$)** 계산
- 50%(반반), 90%(안정권) 획득 지점 시각화 및 커스텀 계산기 제공

### 💰 4. 매출 기대값 분석
- 특정 등급 획득을 위한 **기대 뽑기 횟수 및 현금 가치 환산**
- **천장(Pity) 시스템 시뮬레이션:** 천장 도입 시 매몰 비용 한도 및 유저 구제 비율 분석

### ⚖️ 5. 타 게임 벤치마크 비교
- 메이저 가챠 게임(원신, 블루 아카이브, 니케, 메이플스토리 키우기 등)과 확률 설계 비교
- 장르별(수집형 RPG vs 방치형 RPG) 확률 설계 의도 및 인사이트 도출
